Qu'est-ce ?

IngiGraph est un site web permettant de visualiser les statistiques tirées d'Ingnious, sous forme de graphiques.



Objectif du site :

Ce site à été fait dans le cadre d'un projet (P2) en Sciences Informatique à l'EPL. Son but est de rendre des données, provenant d'une database sqlite, sous forme de graphique avec le framework Chart.js
