IngiGraph

# Présentation

IngiGraph est un site web permettant de visualiser les données relatives aux soumissions Inginious pour 3 cours différents. 
On peut y retrouver 3 types de graphes pour chaque cours :
- Les soumissions en fonction de la tâche / exercice
- Le nombre de soumissions en fonction du résultat pour une tâche (réussi / raté / erreur)
- Le nombre de soumissions sur une semaine

# Lancement

Pour pouvoir accéder au site, il faut tout d'abord avoir installé Flask sur son ordinateur. Ensuite, il faut ouvrir 
le fichier "app.py" et l'exécuter, puis aller à l'adresse http://127.0.0.1:5000/ dans votre navigateur.
